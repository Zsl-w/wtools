# wTools - 快速启动工具

[![Version](https://img.shields.io/badge/version-0.1.0%20beta-blue)](https://github.com/wjunl/wtools/releases)
[![Platform](https://img.shields.io/badge/platform-Windows-blue?logo=windows)](https://github.com/wjunl/wtools/releases)

wTools 是一款 Windows 平台的快速启动工具，灵感来源于 macOS Spotlight。通过快捷键呼出搜索窗口，快速启动应用程序、搜索文件，提升工作效率。

## ✨ 功能特性

- 🚀 **快速启动** - 按 `Alt + Space` 瞬间呼出搜索窗口
- 🔍 **智能搜索** - 支持应用名称、拼音首字母搜索（如输入 `wx` 找到微信）
- 📁 **文件搜索** - 快速查找本地文件（需安装 Everything）
- 🎨 **毛玻璃效果** - 现代化 UI 设计，支持浅色/深色主题
- ⌨️ **键盘操作** - 完全支持键盘导航，无需鼠标
- 🔔 **系统托盘** - 常驻后台，支持开机自启动

## 📥 安装指南

### 方式一：MSI 安装包（推荐）

1. 访问 [Releases 页面]( https://github.com/Zsl-w/wtools/releases)
2. 下载 `wTools_0.1.0_x64_en-US.msi`
3. 双击运行安装程序
4. 安装完成后，按 `Alt + Space` 开始使用

### 方式二：便携版

直接下载 `wTools.exe` 即可运行，无需安装。

## 🎯 使用教程

### 基础操作

| 快捷键 | 功能 |
|--------|------|
| `Alt + Space` | 显示/隐藏搜索窗口 |
| `↑ / ↓` | 切换搜索结果 |
| `Enter` | 打开选中的项目 |
| `Esc` | 隐藏窗口 |
| `右键` | 显示上下文菜单 |

### 搜索技巧

- **应用搜索**：输入应用名称或拼音首字母
  - `wx` → 微信
  - `dd` → 钉钉
  - `vscode` → Visual Studio Code

- **文件搜索**：直接输入文件名关键词（需 Everything 支持）

### 托盘菜单

右键点击系统托盘图标：
- **显示搜索** - 手动呼出搜索窗口
- **开机自启动** - 设置是否随系统启动
- **退出** - 关闭程序

## 🖥️ 系统要求

- Windows 10/11 64位
- 推荐安装 [Everything](https://www.voidtools.com/) 以获得完整文件搜索功能
